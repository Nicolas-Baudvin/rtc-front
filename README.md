# Real Time Chat

## I. Description

RTC est un projet d'appli web qui permet de créer des salons de chat pour discuter en temps réel avec d'autres utilisateurs.
Ce projet a pour but de monter en compétence sur JavaScript.

## II Liens utiles

Repo du back-end: https://github.com/Nicolas-Baudvin/rtc-backend

## III. Technologies utilisées : 

JavaScript, React.js, Node.js, Socket.io, Fastify.js, MongoDB
